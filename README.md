## ✅ Completed Features

- **Full Bootstrap 5.3 redesign** of all 16 original pages
- **Responsive layout** with mobile-friendly navbar (hamburger menu on mobile)
- **Shared nav/footer via JavaScript** (js/nav.js + js/footer.js) — update once, applies everywhere
- **Custom CSS** (css/style.css) with navy + orange color palette, smooth animations, polished card hover effects
- **Live countdown timer** on homepage to Paper Submission Deadline (April 17, 2026)
- **2026 dates & location** updated throughout all pages
- **New submission link** (EasyChair 2026) on all relevant pages
- **Full 2026 committee list** with TPC member emails on committee.html
- **Call to action sections** and sidebar date cards on CFP and Submission pages
- **Password-protected Proceedings** page (password: AsiaJCIS@2026)
- **Bilingual Registration** page (English + Korean tabs)
- **Interactive Program tables** with colour-coded session types
- **Gallery** page with day-by-day photo link cards
